Ord-en-ley

This application is made primarily for lawyers who wish to start their new firm, who can't or don't want to use professional lawyer applications, or simply are open source lovers and want to work with an alternative tool

Download this project as a .zip file Download this project as a tar.gz file

CUSL (Concurso Universitario de Software Libre)

This program is my proposal to the Spanish Open Source University contest.

Now a days the application is under Alpha version, once Alpha-0.0 version is released I will write more information about install and run the program to allow you contribute using issues if you want :)

How to run the program: Ubuntu version: install packages:

sudo apt-get install python-mysqldb mysql-server python-sqlalchemy

Run the unique module to build the DB:

python models/Models.py

To run the app just run this command:

python controllers/controller.py
